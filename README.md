# biblioteca
copia de un proyecto de python en js

esta api es para control de prestamos de una biblioteca, se prestan libros, cables hdmi, alargues, etc.
tiene usuarios autenticados con roles.

como instalar:
1 - instalar node en tu pc.
2 - parado dentro del proyecto ejecutar npm install
3 - arrancar el proyecto con npm run dev
3.1 - frenar con ctrl+c el cuando se cree la bd
4 - para correr los seeders se debe correr npm run seed
5 - ahora si correr el npm run dev
6 - probar rutas
